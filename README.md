

# COCO — Portafolio Artístico

Portafolio web personal mobile-first para Coco, artista escénico con identidad dual: bailarín contemporáneo académico y Drag Queen performer. Construido con HTML5, CSS3 y JavaScript vainilla — sin frameworks, sin dependencias, carga < 2 s.

## Optimización de rendimiento

- Sin frameworks: cero React, Vue, Tailwind ni librerías de animación
- Sin librerías JS: todo el scroll usa IntersectionObserver nativo
- Lazy loading en todos los elementos img
- Google Fonts con display=swap y preconnect para evitar parpadeo de texto
- Variables CSS centralizadas en :root para cambios globales en un punto
- Recomendado: convertir .jpeg a .webp antes de publicar (ahorro ~30%)


- coco-hero.webp: foto hero pantalla completa (formato vertical, alta resolución)
- Conversión de .jpeg a .webp en todos los assets para optimizar carga
- Fotos de galería drag actualizadas si se cambian las imágenes bento



Diseño y desarrollo: portafolio mobile-first optimizado para iPhone 15 / viewport vertical.
